Desarrollo Web

El desarrollo web es el proceso de crear sitios web, incluyendo el diseño, la maquetación, la programación, la implementación de CMS, la creación de contenidos y el mantenimiento de un sitio web.

  • Usamos HTML y CSS para crear páginas
  • Basados en estándares del W3C
  • Usamos JavaScript para crear páginas dinámicas

Implantación y Administración de redes

La implantación y administración de redes es el proceso de crear redes, tanto locales como de área extensa,distribuyendo los recursos de la red entre los distintos usuarios, y administrando dispositivos de red, servidores, etc.

  • Usamos redes cableadas e inalámbricas
  • Basadas en estándares del IEEE
  • La seguridad es nuestra prioridad
